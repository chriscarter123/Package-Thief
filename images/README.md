# Images

Place the following files here to replace placeholder emojis/text:

| File | Used for | Notes |
|------|----------|-------|
| `mascot.png` | Hero section box character | ~140×140px recommended |
| `step-find-box.png` | Step 1 — open cardboard box | ~110×110px |
| `step-load-up.png` | Step 2 — gifts/sparkle | ~110×110px |
| `step-set-bait.png` | Step 3 — box with mask | ~110×110px |
| `qr-code.png` | QR code on "Put that back!" card | 56×56px |

SVG files are also supported — just use the same filenames with `.svg` extension and update `index.html`.
